# Game Programming

Personal game programming project for the games development module.

## Working Title
LexiQuest

## Game Idea
LexiQuest is a small 2D English learning game. The planned vertical slice will focus on one complete level where the player moves through the scene, interacts with English word objects, receives feedback, and reaches a clear win or lose state.

The project is intentionally small in scope so that the final result can be playable, stable, tested, and clearly explained.

## Current Unity Project
- Unity version: 2022.3.62f3c1
- Template: 2D Unity project
- Main project folder: `Assets/`

## Controls
Controls will be updated as the prototype develops.

Planned controls:
- Move: WASD or Arrow Keys
- Interact/Select: Space or Mouse Click
- Restart: R

## How to Run
1. Open Unity Hub.
2. Add or open this project folder.
3. Use Unity 2022.3.62f3c1 or a compatible Unity 2022.3 LTS version.
4. Open the main scene from `Assets/Scenes/`.
5. Press Play in the Unity Editor.

## Project Structure
- `Assets/` - Unity scenes, scripts, prefabs, art, audio, and UI.
- `Packages/` - Unity package manifest and package lock files.
- `ProjectSettings/` - Unity project settings.
- `Docs/` - concept, development plan, testing log, and credits.
- `inclass activity/` - earlier class activity evidence.
